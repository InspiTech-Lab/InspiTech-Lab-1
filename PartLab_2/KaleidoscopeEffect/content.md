# Kaleidoscope Effect

## Snippet Extract

/* --- SNIPPET EXTRACT ---
for (let i = 0; i < segments; i++) {
    ctx.save();
    ctx.translate(centerX, centerY);
    ctx.rotate(segmentAngle * i + autoRotate);

    const gradient = ctx.createRadialGradient(
        this.x - centerX, this.y - centerY, 0,
        this.x - centerX, this.y - centerY, this.size
    );

    ctx.scale(-1, 1);
}
--- END SNIPPET --- */
